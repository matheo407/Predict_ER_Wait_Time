## Description

Ce projet a pour objectif d’analyser les facteurs influençant le temps d’attente des patients aux urgences et de construire des modèles de Machine Learning capables de prédire ce temps d’attente à partir des données patients et contextuelles.

## Structure du projet

ER_Wait_Time_Dataset.csv
requirements.txt
[README.md](http://readme.md/)

```python
pip install -r requirements.txt
```

```bash
jupyter notebook Projet_ML_Alexis_Matheo.ipynb
```

Puis exécuter les cellules dans l’ordre.

## Contenu du projet

- Analyse exploratoire des données
- Prétraitement (nettoyage, encodage, normalisation)
- Entraînement de plusieurs modèles de prédiction du temps d’attente
- Évaluation et comparaison des performances
